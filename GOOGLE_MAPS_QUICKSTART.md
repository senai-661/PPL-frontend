# 🚗 Google Maps - Implementação Rápida

## ⚡ Quick Start (3 Passos)

### 1️⃣ Obter API Key
```
1. Abra: https://console.cloud.google.com/
2. Crie novo projeto
3. Ative APIs: Maps JavaScript, Directions, Places, Geocoding
4. Crie uma API Key em Credentials
5. Copie a chave
```

### 2️⃣ Adicionar a Chave
Edite o arquivo: `/pages/Motorista/dashboard-motorista.html`

Procure pela linha:
```html
<script src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY_HERE&...
```

Substitua `YOUR_API_KEY_HERE` por sua chave real.

### 3️⃣ Pronto! 🎉
Abra a página no navegador e teste!

---

## 🗺️ O Que Funciona

✅ **Mapa Interativo** - Zoom, pan, múltiplos estilos  
✅ **Localização** - Detecta seu GPS automaticamente  
✅ **Rotas** - Calcula melhor rota entre dois pontos  
✅ **Autocomplete** - Sugestões de endereços  
✅ **Instruções** - Passo a passo completo  
✅ **Informações** - Distância, tempo, etc  
✅ **Responsivo** - Funciona em mobile/tablet/desktop  

---

## 🎯 Como Usar

1. Abra o Dashboard do Motorista
2. Clique em **"Calcular Rota"**
3. Digite origem e destino (ou use sua localização)
4. Clique em **"Calcular Rota"**
5. Veja a rota no mapa com instruções!
